## 用自己的话回答

- 什么是特征，什么是标签：特征是模型可以查看并用来做判断的信息，例如用户使用月数、投诉情况和支付方式；标签是希望模型预测的真实答案，本项目中是用户是否流失（Churn）。
- 为什么要保留测试集：测试集模拟模型没有见过的新用户，只在训练与预处理完成后使用，才能较客观地检查模型对新数据的泛化能力，避免只记住训练数据。
- 为什么83%准确率仍可能没有用：数据中未流失用户约占83%，若模型永远预测“不流失”，准确率仍会很高，却一个真实流失用户也找不到，流失召回率为0，无法支持挽留客户的业务目标。
